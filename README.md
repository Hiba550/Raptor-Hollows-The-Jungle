Raptor Hollows : The Jungle (Preview)

The Jungle a dense prehistoric World Gen V2 experience where players read tracks, follow bones, and uncover raptor territory

Raptor Hollows : The Jungle is a focused World Gen V2 exploration pack built around a thick prehistoric jungle. It is meant to feel readable, atmospheric, and playable: follow mud trails, broken plants, bones, claw marks, nests, ruins, and den clues as the jungle slowly reveals raptor territory.

Features
Dense prehistoric jungle generation using existing Hytale assets.
Raptor territory told through tracks, bones, nests, claws, and environmental clues.
Mystery structures, den areas, ruins, fossil landmarks, and hidden loot details
How to Play

Install the release zip in your Hytale mods folder and enable the pack when creating or loading a world.
To directly experience the full Raptor Hollows showcase,

use the World Gen V2 create flow

/worldgen2 create
